## SUMÁRIO EXECUTIVO

**Oportunidade:** Integrar funcionalidades e dados de bynx.gg (portfolio management de Pokémon TCG) no SLABR, transformando a plataforma de um grader/marketplace puro em uma **infraestrutura completa de trading cards do Brasil**.

**Posicionamento:** 
- Bynx.gg = portfolio manager para Pokémon brasileiras (nicho específico)
- SLABR = infraestrutura de confiança + marketplace + múltiplos TCGs (global)
- **Combinação = Cardmarket Brasil** (grading + dados + marketplace + comunidade)

**Recomendação:** Implementar em 3 fases paralelas:
1. **Fase 0 (MVP): Integração leve** — preços bynx no SLABR (4 semanas, R$ 40K)
2. **Fase 1: Integração profunda** — sincronização de coleções + marketplace unificado (16 semanas, R$ 180K)
3. **Fase 2: Plataforma dominante** — expansão para Magic/Yu-Gi-Oh + API/dados (contínua, R$ 200K+/ano)

**ROI esperado em 12 meses:** +150K usuários, +R$ 2M em GMV, +35% LTV

**Banco de Dados - Schema Adicional (Fase 1):**

```sql
-- Usuarios sincronizados com bynx
CREATE TABLE user_bynx_sync (
    id UUID PRIMARY KEY,
    user_id UUID REFERENCES users(id),
    bynx_user_id VARCHAR(255) UNIQUE,
    access_token VARCHAR(1000),
    refresh_token VARCHAR(1000),
    token_expiry TIMESTAMP,
    last_sync TIMESTAMP,
    sync_status VARCHAR(50) DEFAULT 'idle', -- idle, syncing, error
    error_message TEXT,
    auto_sync BOOLEAN DEFAULT TRUE,
    sync_interval_min INTEGER DEFAULT 60,
    created_at TIMESTAMP DEFAULT NOW(),
    updated_at TIMESTAMP DEFAULT NOW()
);

-- Cache de preços com histórico
CREATE TABLE price_history (
    id UUID PRIMARY KEY,
    card_id VARCHAR(100),
    price_brl DECIMAL(10,2),
    price_usd DECIMAL(10,2),
    source VARCHAR(100), -- 'bynx.gg', 'tcgplayer', 'cardmarket'
    date DATE,
    created_at TIMESTAMP DEFAULT NOW(),
    INDEX idx_card_date (card_id, date)
);

-- Histórico de sincronização
CREATE TABLE bynx_sync_log (
    id UUID PRIMARY KEY,
    user_id UUID REFERENCES users(id),
    action VARCHAR(50), -- 'import', 'export', 'price_update'
    status VARCHAR(20), -- 'success', 'error'
    items_processed INT,
    duration_ms INT,
    error_msg TEXT,
    timestamp TIMESTAMP DEFAULT NOW()
);

-- Mapeamento de cartas entre sistemas
CREATE TABLE card_external_ids (
    id UUID PRIMARY KEY,
    slabr_card_id VARCHAR(100),
    bynx_card_id VARCHAR(100),
    cardmarket_id VARCHAR(100),
    tcgplayer_id VARCHAR(100),
    sync_status VARCHAR(50),
    verified_at TIMESTAMP,
    created_at TIMESTAMP DEFAULT NOW()
);
```
